# 🌐 CodeHub Website

This folder contains the source code for the **CodeHub Website**, our group’s central hub.  
The website will showcase our team and projects.

---

## 📖 Overview
The website is being built to:
- Introduce the **CodeHub** community.
- Share our projects (web apps, games, experiments).
- Provide resources, links, and updates about ongoing work.
- Serve as a foundation for future backend development.

---

## 🏗️ Tech Stack
Currently, the website uses:
- **HTML5** – structure  
- **CSS3** – styling  
- **(Planned)** JavaScript – interactivity
